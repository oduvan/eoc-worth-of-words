Вы решили достаточно задач и пришло время немного отдохнуть. Давайте сыграем партию в Scrabble?
У вас есть список слов и ваша задача - определить самое ценное из них. То, которое принесет больше всего очков.
Правила:
Каждое слово имеет ценность, которая эквивалентна сумме очков всех букв, входящих в него.
Стоимость букв следующая:
e, a, i, o, n, r, t, l, s, u = 1
d, g = 2
b, c, m, p = 3
f, h, v, w, y = 4
k = 5
j, x = 8
q, z = 10
Например, ценность слова 'dog' - 5, так как 'd' = 2, 'o' = 1 и 'g' = 2.

Входные данные: Список слов

Выходные данные: Самое ценное слово

Пример:

{% if interpreter.slug == "js-node" %} 
worthOfWords(['hi', 'quiz', 'bomb', 'president']) == 'quiz'
worthOfWords(['zero', 'one', 'two', 'three', 'four', 'five']) == 'zero'
{% else %} 
worth_of_words(['hi', 'quiz', 'bomb', 'president']) == 'quiz'
worth_of_words(['zero', 'one', 'two', 'three', 'four', 'five']) == 'zero'
{% endif %}

Как это используется: Для анализа текстов.

Предусловия:
2 <= количество слов <= 10
во всех словах используются только буквы нижнего регистра
